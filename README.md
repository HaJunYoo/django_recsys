## Django Musinsa Recommendation 

### 개요
- word2vec을 이용한 컨텐츠 based 추천
- 총 상품 9500개
- 리뷰 정보와 컨텐츠 정보들을 가지고 word2vec 임베딩 수행

### Stack

  - Bootstrap
  - word2vec => (npy 파일 이용)
  - pandas
  - sqlite
